Programming Languages

C
C is an imperative procedural language, supporting structured programming, lexical variable scope, and recursion, with a static type system.

C++
C++ is a cross-platform language that can be used to create high-performance applications.

Javascript
JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Python
Python is commonly used for developing websites and software, task automation, data analysis, and data visualization.

Typescript
TypeScript allows specifying the types of data being passed around within the code, and has the ability to report errors when the types don't match.
